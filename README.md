```
D. Petrinovic, "Causal Cubic Splines: Formulations, Interpolation Properties and Implementations," in IEEE Transactions on Signal Processing, vol. 56, no. 11, pp. 5442-5453, Nov. 2008, doi: 10.1109/TSP.2008.929133.
```
